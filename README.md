# Congregación La Dolorosa del Colegio

Este es un proyecto web elegante y devocional dedicado a la **Parroquia La Esperanza - Congregación La Dolorosa del Colegio**. La página presenta un diseño solemne con efectos visuales modernos, optimizado para una experiencia espiritual e informativa.

## ✨ Características

- **Diseño Solemne:** Estética basada en colores azul profundo y dorado, inspirada en la iconografía de la Virgen.
- **Efectos Visuales:** Sistema de partículas animadas en el fondo mediante Canvas API.
- **Animaciones de Entrada:** Revelación suave de elementos al hacer scroll (Scroll Reveal).
- **Diseño Responsivo:** Optimizado para dispositivos móviles y escritorio.
- **Interacción:** Integración directa con WhatsApp para contacto rápido.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica.
- **CSS3:** Estilos avanzados, Flexbox, y animaciones personalizadas.
- **JavaScript (Vanilla):** Sistema de partículas y lógica de animaciones.
- **Google Fonts:** Uso de tipografías clásicas como *Cinzel* y *Cormorant Garamond*.
